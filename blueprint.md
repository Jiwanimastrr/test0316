
# Lotto Number Generator

## Overview

A simple, visually appealing web application to generate random lottery numbers. The application will feature a clean, modern design and provide an intuitive user experience.

## Project Outline

*   **index.html**: Main HTML structure, including the title, a container for the lottery numbers, and a button to generate them.
*   **style.css**: Modern styling for the application, including a responsive layout, custom fonts, and a visually appealing color scheme.
*   **main.js**: JavaScript logic to generate unique, random lottery numbers and display them in the UI.

## Current Task: Initial Setup

1.  **Modify `index.html`**: Set up the basic structure for the lottery number generator.
2.  **Modify `style.css`**: Add styles to create a modern and visually appealing UI.
3.  **Modify `main.js`**: Implement the lottery number generation logic.
